Instead of a button, a slider is used to apply a brightness filter on the image. The slider has a value from 0 to 200. The Javascript code sets a brightness filter to the image in the canvas using the value of the slider and a percentage. For example, the value of the slider in the middle is 100. This will set the brightness filter to 100%, making the image appear normal.
