RGB

 

Change Red, Blue, Green Values

 

On this page, a file button on the top allows the user to choose any image that they have downloaded. Below the images are several buttons that change every individual pixel in the image. The Make Red button will set every pixel in the image to have a red value of 255, the Make Blue button sets every pixel to a blue value of 255, and the Make Green button sets every pixel to a green value of 255. The user can make different color filters by pressing different buttons together. The reset button will reset the image so that the color filter will disappear.
