Grayscale

 

Make Gray

 

Again, there is a file button that allows the user to upload any image they have on their computer. The Make Gray button below the image will simply apply a grayscale effect to the image. The Make Gray button is connected to a Javascript function that finds the average of the red, green, and blue values of an individual pixel. The red, green, and blue value of the pixel is then altered to be the average value from earlier. A for loop is used so that the average of every pixel is found and the rgb values of the pixels are all replaced.
